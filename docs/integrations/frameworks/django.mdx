---
title: "Django"
description: "How to use KMS to inject environment variables and secrets into a Django app."
---

Prerequisites:

- Set up and add envars to [KMS Cloud](https://kms.hanzo.ai)
- [Install the CLI](/cli/overview)

## Initialize KMS for your [Django](https://www.djangoproject.com) project

```bash
# navigate to the root of your of your project 
cd /path/to/project

# then initialize KMS
hanzo-kms init
```

## Start your application as usual but with KMS

```bash
hanzo-kms run -- <your application start command>

# Example
hanzo-kms run -- python manage.py runserver
```
