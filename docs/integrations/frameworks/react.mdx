---
title: "React"
description: "How to use KMS to inject environment variables and secrets into a React app."
---

Prerequisites:

- Set up and add envars to [KMS Cloud](https://kms.hanzo.ai)
- [Install the CLI](/cli/overview)

## Initialize KMS for your [Create React App](https://create-react-app.dev)

```bash
# navigate to the root of your of your project 
cd /path/to/project

# then initialize kms
hanzo-kms init
```

## Start your application as usual but with KMS

```bash
hanzo-kms run -- <your application start command>

# Example
hanzo-kms run -- npm run dev
```

<Info>
  React environment variables must be prefixed with `REACT_APP_` to show up within the application 
</Info>
