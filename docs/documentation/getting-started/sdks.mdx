From local development to production, KMS's language-specific SDKs provide the easiest way for your app to fetch back secrets on demand.

Prerequisites:

- Have a project with secrets ready in [KMS Cloud](https://kms.hanzo.ai).
- Create an [KMS Token](/documentation/platform/token) scoped to an environment in your project in KMS.

## Usage

Follow the instructions for your language use the SDK for it:

- [Node.js SDK](https://kms.hanzo.ai/docs/sdks/languages/node)
- [Python SDK](https://kms.hanzo.ai/docs/sdks/languages/python)
- [Java SDK](https://kms.hanzo.ai/docs/sdks/languages/java)
- [.NET SDK](https://kms.hanzo.ai/docs/sdks/languages/csharp)
- [Go SDK](https://kms.hanzo.ai/docs/sdks/languages/go)
- [C++ SDK](https://kms.hanzo.ai/docs/sdks/languages/cpp)
- [Ruby SDK](https://kms.hanzo.ai/docs/sdks/languages/ruby)

Missing a language? [Throw in a request here](https://github.com/hanzoai/kms/issues).
