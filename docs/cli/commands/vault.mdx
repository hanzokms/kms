---
title: "hanzo-kms vault"
description: "Change the vault type in KMS"
---

<Tabs>
  <Tab title="View current Vault">
    ```bash
    hanzo-kms vault 

    # Example output
    Vaults are used to securely store your login details locally. Available vaults:
    - auto (automatically select native vault on system)
    - file (encrypted file vault)

    You are currently using [file] vault to store your login credentials
    ```
  </Tab>

  <Tab title="Switch vault">
    ```bash
    hanzo-kms vault set <name-of-vault>

    # Example 
    hanzo-kms vault set keychain
    ```
  </Tab>
</Tabs>


## Description

To safeguard your login details when using the CLI, KMS attempts to store them in a system keyring. If a system keyring cannot be found on your machine, the data is stored in a config file.
